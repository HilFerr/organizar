TERMINAL KEYWORDS:

	'javac Main.java'

	COMPILES THE DOCUMENT;

	'javac' - COMPILES THE CODE;
	'Main.java' - FORMAL FILE NAME;

	---------------------
	|    'java Main'    |
	---------------------

	EXECUTES THE CODE;

	'java' - FILE TYPE;
	'Main' - INFORMAL FILE NAME;

CODE KEYPOINTS:

----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
	'System.out.print()' - Prints whatever is put inside of the parentisis.
	'System.out.println()' - Does the same of the previous one but outputs and 'enter' in the end, making so if there's another 'System.out.println()' it will be written on the next line.
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

	Example:

	System.out.println("A Toyota Corolla...");
	System.out.print("Made in 2005 ");
	System.out.print("is a bad personal vehicle in 2024.");

----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

	Outputs:

	A Toyota Corolla...
	Made in 2005 is a bad personal vehicle in 2024.

==============================================================================================================================================================================================

----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
	'x.indexOf(y)' - Checks if the contents inside of 'x' are inside of the variable 'y' and returns the location of the character where it starts.
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------	

	Example:

		x = "Toyota Corolla";
		y = "Corolla";

		int checksContents = x.indexOf(y);

		System.out.println(checksContents);
		
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

	Outputs:

	7 // Because that is the location where the word 'Corolla' starts.
